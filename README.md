# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/patro-krishna/CareConnect.git
```

### Go to Project Folder

```bash
cd CareConnect
```

### Install Dependencies

```bash
npm install
```

### Create .env File

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

### Run Development Server

```bash
npm run dev
```

### Production

```bash
npm start
```
